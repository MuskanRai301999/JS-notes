What is JavaScript
1.High-level-cross platform, memory management.
2.Light weighted -Low uses of system resources
3.Interpreted - Execute instructions directly without earlier compiling a program into machine language line by line
4.JIT compiled- Compiles JavaScript to executable bytecode just as it is about to run .. at the runtime
5.Multi-paradigm - different styles to write the code
6.Prototype-based - Allows the creation of an object without first defining its class.
7.Single-threaded - Executes one line of code at a time/one task at a time
8.Dynamically-typed Language-capable of change at runtime
